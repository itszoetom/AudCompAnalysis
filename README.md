## Repository Structure

### **Evoked**
- **Discriminability** — Measures how well neural responses distinguish between auditory stimuli using metrics like d-prime and LDA.
- **PCA** — Dimensionality reduction and visualization of population neural activity, calculating participation ratio.
- **Ridge** — Decoding of stimulus features from neural activity using Ridge Regression.

### **Single Mouse**
- Plots PCA and spiking analyses for individual mice across dates, conditions, and brain areas.

### **SpikeWindows**
- Time-windowed analyses of discriminability, PCA participation ratio, and Ridge regression.
